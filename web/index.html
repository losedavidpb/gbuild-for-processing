gBuild

A Library by David Parreño Barbuzano for Processing
Last update: 14/03/2022.

Tools to construct complex UI components using primitives.

Download

Download gBuild version 1.0.0 (1) in project at Github.

Installation

Unzip and put the extracted gbuild-for-processing folder into the libraries folder of your Processing sketches. Reference and examples are included in the gBuild folder.

Keywords. processing.org, library, ui, graphical-component, ui-design, processing-library

Reference. Have a look at the javadoc reference.

Source. The repo of gBuild can be browsed here.
